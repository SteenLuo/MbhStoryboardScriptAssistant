# 对话学习记录模板

## 一、是否需要学习

| 项目 | 内容 |
| --- | --- |
| 是否需要学习 | 是 / 否 / 待判断 |
| 材料类型 | 小说 / 剧本 / 分镜 / 反馈 / 质量问题 / 流程偏好 / 普通问答 |
| 质量信号 | 变好 / 变差 / 无明显变化 / 待判断 |
| 学习动作 | 候选规则 / 降质记录 / 更新快照 / 跳过 / 待评测 |

## 二、可学习内容

简述本次对话中可沉淀的规则、偏好、限制或反例。

## 三、证据

记录用户原始意思，不要夸大，不要把单次偏好误判为普适规则。

## 四、下一步

说明进入候选规则、等待评测、直接跳过，或触发降质回退。
