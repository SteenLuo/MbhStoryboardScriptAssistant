# M0-M6 网页端完整性对照表

日期：2026-06-11

## 一、结论

M7 网页入口现在已经从“只保留提示词原则”升级为“真实接入本地技能和无感学习闭环”。

当前已接入的关键能力：

- 网页对话会根据用户输入路由到本地 `skills/`。
- 每轮只读取命中的 `SKILL.md` 和对应 `references/*.md`，不全量读取整个技能目录。
- 助手消息会保存 `skillRoute`、`usage` 和学习记录路径。
- 出现偏好、规则、质量反馈、M0-M6 完整性、skill 和无感学习机制等信号时，会自动写入 `learning/conversation-records/`。
- `tools/Invoke-AutoLearningCycle.ps1` 已验证能消费网页自动生成的对话学习记录，并生成候选规则草案、学习快照和技能进化草案。
- 设置面板已增加 M0-M6 完整性状态和“刷新学习闭环”入口。
- M8 第一段已把 M3 阶段化工作台改为流程思维导图，可选择当前对话或历史运行记录，点击节点查看产物，并进行手动归档学习。
