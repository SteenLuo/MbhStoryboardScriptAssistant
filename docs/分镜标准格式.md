# 分镜标准格式

本文档记录当前项目的 AI 漫剧分镜标准格式。后续生成分镜时，应优先使用此格式。

本格式默认服务 AI 漫剧生产，重点不是传统拍摄现场调度，而是让每个镜头都能继续转成 AI 画面生成、AI 视频生成、字幕、音效和剪辑节拍。

## 一、场次格式

每个场次以场次信息开头：

```text
场次：1-1
地点：无
时间：无
人物：无
```

字段说明：

- `场次`：建议使用“集数-场次”格式，例如 `1-1`、`1-2`。
- `地点`：写明场景地点；如果没有明确地点，写 `无`。
- `时间`：写明时间和内外景，例如 `日 外`；如果没有明确时间，写 `无`。
- `人物`：写本场出现的人物；如果无人，写 `无`。

## 二、镜头格式

每个镜头按以下字段书写：

```text
镜号：1
景别：远景
运镜：缓推
情绪/动作：无垠星海之中，无数个由法则锁链缠绕的白色光团沉浮不定。
音效：星空BMG
台词：旁白VO：太初混沌，万界初开。
时长：3s
```

标准字段：

- `镜号`
- `景别`
- `运镜`
- `情绪/动作`
- `音效`
- `台词`
- `时长`

可选字段：

- `字幕`

## 三、字段顺序

推荐顺序：

```text
镜号：
景别：
运镜：
情绪/动作：
音效：
台词：
时长：
字幕：
```

如果原始材料中 `台词` 和 `音效` 顺序不同，也可以接受；但新生成分镜时建议统一为上面顺序。

## 四、写法要求

### 场次

同一个场次下可以包含多个镜头。进入新场次时，重新写：

```text
场次：
地点：
时间：
人物：
```

### 镜号

镜号应连续编号。跨场次后，镜号可以继续累加，不必从 1 重新开始。

### 景别

常见写法包括：

- 远景
- 全景
- 中景
- 近景
- 特写
- 面部特写
- 俯拍特写
- 仰拍中景
- 过肩中景
- 全景-中景

景别可以包含拍摄角度，例如 `仰拍中景`、`俯拍侧面特写`。

### 运镜

常见写法包括：

- 固定
- 缓推
- 快速推进
- 缓慢平移
- 下摇
- 下移
- 跟镜头
- 仰拍固定
- 俯拍固定
- 下移跟随
- 拉
- 摇
- 移
- 升格
- 降格
- 手持
- 甩镜头
- 晃镜头

如果运镜暂时没有明确设计，可以留空，但生成新分镜时应尽量补齐。

运镜原则：

- 不要全是固定镜头。
- 不要为了变化而乱运镜。
- 推用于逐渐聚焦角色或物品。
- 拉用于逐渐展示角色周围环境。
- 摇用于角色第一人称视角观察四周环境或人物。
- 移用于机位从一个位置平移到另一个位置。
- 升格用于机位从低视角往高视角上升。
- 降格用于机位从高视角往低视角下降。
- 手持、甩镜头、晃镜头用于慌张、混乱、突发反应等情绪，但不要滥用。

### 构图

人物对话不要过多使用正面怼脸。可根据剧情使用正面、正四三、正反打、外反打、内反打、前景遮挡等构图。

单个人物说台词时，优先使用近景或中景，少用特写。不要频繁对同一角色做多角度、多景别切换。

需要切镜时，可以切到其他角色反应、身体部位反应或相关物品画面；如果没有合适反应，不要硬切。

画面视角不要全是平视。可根据内容使用平视、仰视、俯视、低视角、地面视角或特殊视角。

### 情绪/动作

这是分镜中最重要的字段，应写出具体画面，而不是抽象描述。

推荐写法：

- 写清楚画面主体。
- 写清楚人物动作。
- 写清楚表情和情绪。
- 写清楚关键道具或特效。
- 必要时用括号补充前景、背景、特效说明。

示例：

```text
情绪/动作：夏清站在人群前方，神情高傲，背景人群神情激动互相交谈。
```

### 台词

台词可以包含：

- 人物对白
- 旁白VO
- 系统音VO
- 画外音
- 人物内心 OS

示例：

```text
台词：旁白VO：太初混沌，万界初开。
台词：（画外音）我不是刚氪了10个648好不容易出了一个金吗？
台词：凌天尊者OS：此女灵运极强，天赋惊人。
```

### 音效

音效可以写背景音乐、环境声、特效声。

示例：

```text
音效：星空BMG
音效：星空BMG＋破碎
音效：熙熙攘攘的人群
```

没有音效时可以留空：

```text
音效：
```

### 时长

统一使用秒数，例如：

```text
时长：3s
时长：4s
时长：6s
```

## 五、标准模板

```text
场次：
地点：
时间：
人物：

镜号：
景别：
运镜：
情绪/动作：
音效：
台词：
时长：
字幕：
```

## 六、生成分镜时的要求

后续由剧本生成分镜时，应遵守：

1. 先写场次信息，再写镜头。
2. 每个镜头必须有 `镜号`、`景别`、`运镜`、`情绪/动作`、`音效`、`台词`、`时长`。
3. `情绪/动作` 必须是具体画面。
4. `台词` 要保留旁白、画外音、系统音等标记。
5. `时长` 使用 `3s`、`4s` 这类格式。
6. 如果需要字幕，放在镜头字段末尾。
7. 不使用表格作为默认输出格式，默认使用逐镜头文本块。
8. 默认按 AI 漫剧用途生成，画面主体、动作、情绪、字幕和音效要清楚。
9. 避免在单个镜头里堆叠过多主体、动作和复杂空间关系，降低后续 AI 生成难度。
