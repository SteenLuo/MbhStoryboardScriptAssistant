## 1. 总体结论

本次评审对象为修改后的 60 集剧本《八零杂货铺完本.docx》。本报告主要评估剧本内部硬伤、表达可执行性，以及是否具备进入短剧/漫剧分镜和制作的基础；不评估市场热度、平台投放或商业卖点。

修改稿已经修掉旧版中较明显的后段换题材问题，例如“天霸集团、股权激励、发布会、法院/警局、海边求婚、报童号外”等内容在本版正文中未检索到。整体主线从“空间起家”逐步转向“货源正规化、县城成衣、乡村带动”，方向比旧版统一。

但本版仍不建议直接进入分镜或制作。主要原因是：核心合法化链条仍有跳步，大棚检查仍靠造假式应急通过，空间规则和玉佩代价没有完全闭环，后 57-60 集扩张为服装厂、大棚基地和全村跃迁的速度过快，人物线与伏笔仍有残留未兑现。建议先做一轮定向返修，而不是整体推翻重写。

## 21. 返修验收标准

修改稿至少应满足以下标准：

| 验收项 | 通过标准 |
| --- | --- |
| 集数格式 | 60 个独立集标题，标题样式统一，无粘连 |
| 剧本格式规范 | 场景标题、人物行、△画面段、人物台词/动作、旁白/VO/OS/内心独白归属清楚；动作和说出口的台词不能混成无法解析的一句 |
| 人物资产 | 小传中出现的人物，正文必须出场并完成叙事功能；不用则删除 |
| 货源合法化 | 明确过去库存如何处理，未来货源如何进账，工商备案如何完成 |
| 空间规则 | 明确每日次数、每次体量、库存消耗、玉佩裂纹后果 |
| 大棚检查 | 不再靠“埋成品菜”无代价过关，或让主角因此付出明确代价 |
| 广州线 | 灰衬衫/李经理手下必须有明确行动和后果 |
| 人物转变 | 王婶、张主任、孙志强的转变都有事件驱动和情绪过程 |
| 后段扩张 | 建厂、大棚五亩、村庄变化都有手续、资金、人员、时间过程 |
| 年代常识 | 婚姻登记、长途电话、介绍信/住宿、乡镇企业术语完成核查或规避 |
| 生产用工 | 质量问题应有培训、质检、返工、薪酬和留人制度，不应简单用扣工钱解决 |
| 商务出差 | 出差台词应服务谈判、验货、付款、托运和安全，不应出现与任务相反的幼稚化叮嘱 |
| 语言表达质量 | 硬表达问题完成修正：关键追问不能靠主观保证过关，商务场景话术必须服务当前任务，称谓承接一致，心理信息用 OS/VO + 可视画面承托 |
| 蔬菜供应 | 国营饭店采购、送货、结账、货源口径必须统一可查 |
| 门店扩张 | 新店开业和广州出差之间必须有店长、库存、收银和交接机制 |
| 质量召回 | 召回退款必须有可执行的信息来源，如小票、登记、熟客或单位名单 |
| 对话称谓 | 同一场戏内称谓、人物已知信息和上一句台词必须能承接，不能自相矛盾 |
| 画面可呈现 | 心理判断可用内心独白/OS/VO表达，但必须配可视化画面；抽象总结、主题口号必须有具体场面、道具或冲突结果承托 |
| 超自然画面 | 空间、玉佩、裂纹、取物必须有稳定视觉规则，不能只写“意识”“仿佛”“像是在回应” |
| 价值观 | 结尾“踏实努力”有具体行动和代价闭环支撑；角色自辩台词可以保留，但不能被剧情当作无手续过关的依据 |
