# 2026-07-01 画布项目与版本合并更新说明

本说明记录 2026-07-01 交付版的主要更新，面向内部回溯、客户更新沟通和打包交付检查。

## 版本标识

- 前端静态资源版本：`20260701-canvas-merge-rail-v1`
- 客户更新包目标：`dist/MbhStoryboardScriptAssistant-CustomerTrial-NoSkillOverwrite-v*.zip`，以本次打包脚本输出为准。
- 包类型：无 `skills/` 更新包，不覆盖客户已有技能目录。

## 主要更新

### 1. 对话项目管理

- 左侧对话列表按项目分组展示，项目支持展开、收起和重命名。
- 新建对话改为弹窗流程，可选择已有项目或创建新项目。
- 顶部工具栏改为紧凑图标布局，支持搜索、侧边栏收起和侧边栏宽度拖拽。
- 收起侧边栏后只保留展开按钮，减少画布和对话区域占用。

### 2. 自由画布基础能力

- 增加对话 / 画布工作模式切换。
- 画布支持新建、重命名和搜索；搜索结果可定位到命中的节点文本。
- 画布节点支持小说、剧本、分镜脚本、标识四类，并使用对应图标区分。
- 节点内容可直接在框内编辑；长内容支持自定义配色滚动条。
- 支持左键拖拽节点、中键拖拽画布、缩放、适配全部元素、定位选中节点和缩略图拖动。
- 移除画布底部节点统计栏，改为左下角视图工具承载缩放和定位操作。

### 3. 画布连线和生成交互

- 节点加号承担后续生成和手动连线入口；右键菜单不再包含连接项。
- 左键短按加号打开生成菜单，长按可拖出连线并连接到目标节点左右侧。
- 连线可选中，支持添加注释和删除，并使用项目配色的确认弹窗。
- 小说节点右侧可生成剧本和标识；剧本节点右侧可生成分镜脚本和标识。
- 生成分镜脚本时按本地分镜 skill 规则执行，并按集数拆分为多个分镜节点。
- 分镜生成时间较长时，源节点后方展示转圈加载反馈。

### 4. 修改节点

- 小说、剧本、分镜脚本节点的加号菜单增加“修改”能力。
- 修改会生成同类型子节点，并在节点下方显示一次性对话框。
- 修改节点只能绑定一个父级节点；对话内容会随父级内容一起发送给 API。
- 对话生效后输入框锁定为只读，避免重复提交。
- 修改节点右上角显示“修”标识，用于区分普通生成节点和可对话修改节点。

### 5. 合并版本

- 支持框选多个同类型节点打组，并指定唯一版本作为对外对接版本。
- 合并节点保留所选节点的历史版本、父级来源、对话内容和返回信息。
- 点击右上角“合”标识可打开历史版本弹窗，并可重新指定唯一版本。
- 合并节点正文只显示当前唯一版本内容，不再在框内展示历史说明。
- 合并节点视觉已改为顶部贴边版本轨道：当前唯一版本高亮，不额外占用正文空间，也不遮挡节点加号。

### 6. 视觉和可用性修正

- 深色模式下画布顶部栏、工具按钮和滚动条重新适配配色，提高可读性。
- 节点、历史弹窗、对话框和确认弹窗统一使用项目配色的滚动条和圆角风格。
- 画布合并节点的旧堆叠阴影效果已移除，避免层级感不清和遮挡加号。
- `app/public/prototypes/` 仅作为内部样式评审页面，已从客户包中排除。

## 交付影响

- 已有客户 `skills/` 不会被无 skill 更新包覆盖。
- 已经生成的下游内容不会因为重新指定合并唯一版本而自动改写。
- 画布内已有名称冲突不会自动重命名；后续改名时会提示并阻止重复名称。
- 运行数据、API Key、本机配置、历史对话和运行日志仍由打包脚本排除。

## 打包方式

生成无 skill 更新包：

```powershell
powershell -ExecutionPolicy Bypass -File .\tools\New-CustomerTrialPackage.ps1 -Mode NoSkillOverwrite
```

打包脚本会执行干净包校验，确保更新包不包含：

- `skills/`
- `.env` / `.env.local`
- 本机模型配置
- 运行数据和对话记录
- `app/runtime/`
- `runs/`
- 内部原型页 `app/public/prototypes/`
