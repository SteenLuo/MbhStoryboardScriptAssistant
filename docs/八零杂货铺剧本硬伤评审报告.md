# 《八零杂货铺完本》剧本硬伤评审报告

评审口径：使用项目 Skill `skills/02-script/script-hard-issue-review`。本报告只评审剧本内部硬伤、逻辑漏洞、伏笔兑现、人物连续性和返修风险；不评估市场前景、平台适配或漫剧改编价值。

## 1. 总体结论

当前稿件不建议直接进入后续制作或分镜环节。主要问题不是单集台词粗糙，而是第 31 集后发生结构级换轨：前 30 集是“1982 年农村女性创业 + 空间货源 + 供销社压力 + 货源合法化危机”，后 30 集突然变成“县城/省城公司化商战 + 总裁反派 + 发布会反杀”。前半段建立的核心风险没有被消化，后半段大量靠新地图、新反派、新资源解决冲突，导致全剧像两套剧本拼接。

建议判断：结构级重写。优先保留第 1-30 集中比较有效的年代生活质感、空间秘密、供销社压力、王婶/孙志强/陈建军的怀疑线；第 31-60 集需要围绕前半段已埋伏笔重构，不宜继续沿用省城集团、股权激励、股价、发布会、五百万索赔、法院/警局等现代商战表达。
