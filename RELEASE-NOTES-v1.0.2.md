# v1.0.2 更新说明

## 修复

- 修复客户解压新全量包后，浏览器仍连接到旧服务进程，导致“不能创建新画布”的问题。
- 启动助手现在会读取当前端口服务的安装目录；如果 17877 上运行的是旧包或未知包，会提示先从当前包执行重启服务或关闭旧服务窗口，不再静默打开旧页面。

## 使用建议

- 给客户发送 `MbhStoryboardScriptAssistant-CustomerTrial-Full-v1.0.2.zip`。
- 客户解压后优先运行根目录里的 `重启服务.bat`，确保旧服务被关闭后再启动当前包。
- `NoSkillOverwrite` 包不能作为独立安装包使用，缺少本地技能目录时无法完整运行。
