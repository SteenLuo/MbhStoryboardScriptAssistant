# 待补样例清单

日期：2026-06-10

## 一、用途

本文件记录当前项目为了升级题材专属规则，最需要补充的样例类型。

使用人员仍然不需要整理文件。只要把材料放进 `samples/_inbox/`，或直接在对话窗口发送，系统会自动分拣。

## 二、当前最缺的样例

### 1. 经营逆袭类

目标：验证 C-006 经营逆袭镜头链。

最好补充：

- 非八零杂货铺的经营 / 摆摊 / 开店 / 电商 / 直播带货样例。
- 有剧本和分镜最好；只有剧本或只有分镜也可以。
- 最好包含“商品展示、被质疑、成交、打脸、收益反馈”中的多个环节。

为什么需要：

- 当前证据主要来自八零杂货铺。
- 八零第二集、第三集都属于部分对应样例，不能独自支撑正式题材规则。

### 2. 身份反转 / 打脸类

目标：验证 C-007 身份反转镜头链。

最好补充：

- 隐藏老板。
- 隐藏富豪。
- 马甲掉落。
- 穷小子 / 灰姑娘真实身份反转。
- 被看不起后被权威人物认出。

为什么需要：

- 当前强证据主要来自第十七集。
- 需要判断“豪车、列队、包场”是否只是第十七集特定表达，还是身份反转通用表达。

### 3. 修仙 / 玄幻类

目标：验证 C-010、C-012、C-013。

最好补充：

- 非欧皇盲盒的修仙 / 玄幻样例。
- 废柴测试、灵根觉醒、宗门大比、宝物识别、老祖出关、功法反转等片段。

为什么需要：

- 当前修仙证据来自欧皇同一部剧。
- 欧皇是“系统 + 盲盒 + 欧皇”路线，不一定代表所有修仙题材。

### 4. 豪门替嫁 / 强女主类

目标：验证 C-014、C-015。

最好补充：

- 替嫁。
- 真假千金。
- 退婚。
- 马甲女主。
- 豪门回归。
- 强女主冷静反击或高压反击。

为什么需要：

- 当前证据主要来自司爷少奶奶第一章。
- 需要判断强女主反击是否都适合“冷静压迫”风格。

### 5. 信息层边界样例

目标：细分 C-002、C-011 的适用边界。

最好补充：

- 直播弹幕类。
- 系统面板类。
- 手机聊天 / 转账 / 定位类。
- 修仙功法 / 宝物 / 属性面板类。

为什么需要：

- 信息层已经作为基础能力部分确认。
- 但不同信息层的密度和镜头写法还需要细化。

## 三、放入材料后系统要做什么

新材料进入 `samples/_inbox/` 后，系统按以下流程处理：

```text
扫描新文件
-> 按文件名和正文识别剧目
-> 判断小说 / 剧本 / 分镜 / 混合材料
-> 按一部剧归拢
-> 按集数建立 episodes
-> 判断完整强对应 / 部分对应 / 观察样例
-> 生成 notes.md
-> 更新样例报告
-> 判断是否补足某个题材规则证据
-> 必要时生成评测任务
```

## 四、触发评测的条件

新样例满足以下任一条件时，应进入 M4 评测：

- 属于当前缺口题材。
- 有剧本和分镜完整对应。
- 能补足某个候选规则证据。
- 用户明确说这是认可样例。
- 与现有规则冲突，需要校验边界。

## 五、暂不触发评测的情况

- 无法判断剧目归属。
- 只有碎片，且缺少人物和剧情上下文。
- 明显是废稿或不满意稿。
- 与已整理样例重复，但版本关系不清。

这些材料先进入待确认清单，不直接进入规则升级。
