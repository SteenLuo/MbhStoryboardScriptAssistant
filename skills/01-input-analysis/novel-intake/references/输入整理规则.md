# 输入整理规则

## 材料类型判断

- 小说：以叙述段落为主，有心理描写、环境描写、非剧本化对白。
- 剧本：按场景、对白、动作、旁白推进。
- 分镜：按镜头编号、景别、画面、角色动作推进。
- 混合材料：同时包含小说原文、剧本或分镜。

## 初始类型桶

- 都市情感
- 古风情感
- 悬疑惊悚
- 推理
- 玄幻
- 仙侠 / 修仙
- 科幻
- 家庭伦理
- 职场故事
- 复仇 / 逆袭
- 喜剧
- 青春 / 校园

## 必问缺口

只有在缺口会影响下一步结果时才提问：

- 目标平台不明。
- 目标时长或字数不明。
- 剧本是否允许大改不明。
- 分镜是否需要详细到镜头级不明。
- 用户是否已有认可样例不明。

默认目标平台是 AI 漫剧。只有用户明确指定传统影视、广播剧、漫画脚本或其他用途时，才切换判断标准。

## 正向分析规则

拿到小说或剧本后，先分析故事成立的正向结构：

- 小说总纲。
- 每集大纲。
- 每集爽点或剧情高潮点。

每集爽点 / 高潮点：

- 最少 2 个。
- 最多 4 个。
- 少于 2 个，标记“高潮不足”。
- 多于 4 个，标记“节奏可能过密”。

常见爽点 / 高潮点：

- 反转。
- 打脸。
- 危机解除。
- 身份揭露。
- 情感爆发。
- 战力展示。
- 关键道具出现。
- 冲突升级。
- 悬念抛出。

## 负面分析规则

必须同时检查单集剧情和整体剧情。

重点检查：

- 世界观设定矛盾。
- 人物行为不符合性格、身份、动机或处境。
- 因果关系缺失或跳跃。
- 空间错乱。
- 时间顺序混乱。
- 人物状态遗忘。
- 人物生死矛盾。
- 伤病、剧毒、能力、身份前后矛盾。
- 知识或常识性错误。
- 对话与信息逻辑混乱。
- 伏笔未回收。
- 重大事件被遗忘。
- 上一集冲突未处理，下一集直接跳走。

典型问题：

- 上一秒在大街上，下一秒在卧室，没有转场或解释。
- 人物中毒后，后续完全不提毒。
- 人物已经死亡，续集突然出现，既不是回忆，也没有伏笔或解释。
- 角色突然拥有此前没有的能力，但没有成长、道具或解释。
