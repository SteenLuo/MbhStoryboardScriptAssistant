# v1.0.1 更新说明

本次更新重点修复画布一键分镜在客户实际使用中暴露的两个问题。

## 修复内容

- 修复画布剧本节点一键分镜后，原剧本内容可能被清空的问题。
- 修复剧本分集标题被编辑器保存为 Markdown 加粗格式后，只能识别出一集分镜的问题。
- 支持从中间集开始的剧本内容，例如第 7 集到第 13 集，也能按实际集数拆分。
- 分集识别会清理标题行中的 Markdown 标记，避免 `**第一集： **` 这类格式污染后续分镜生成。
- 本地静态资源禁用浏览器缓存，并更新前端资源版本标识，减少客户更新后仍加载旧脚本的情况。

## 验证结果

- 当前客户剧本样例可识别为 3 集：第 1 集、第 2 集、第 3 集。
- 全量自动测试通过：263 项通过，0 项失败。

## 包类型

本次建议使用更新包：

- `MbhStoryboardScriptAssistant-CustomerTrial-NoSkillOverwrite-v1.0.1.zip`

该包不覆盖客户已有 `skills/` 目录。
